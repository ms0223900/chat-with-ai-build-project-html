# chat-with-ai-build-project-html

## 繁體中文

本專案用來**集中放置**與課程相關的 **demo 網頁**（主要是靜態 HTML）。  
內容多為課堂示範、練習或小專案頁面，方便備份、版本管理與日後對照。

### 如何使用

- 以瀏覽器直接開啟各 `.html` 檔即可預覽（或透過本機簡易 HTTP 伺服器服務整個資料夾）。

---

## English

This repository is a **home for course-related demo webpages**, mostly static HTML.  
It holds in-class demos, exercises, and small project pages for backup, versioning, and future reference.

### How to use

- Open any `.html` file in a browser for a quick preview, or serve the folder with a simple local HTTP server.
